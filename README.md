# Restan - Restaurant Website

A modern, elegant restaurant website built with **HTML** and **Tailwind CSS** - featuring a beautiful design with smooth animations and responsive layout.

## 🌟 Features

- **Modern Design**: Clean and professional interface with golden accents
- **Responsive Layout**: Fully optimized for all devices (desktop, tablet, mobile)
- **Tailwind CSS**: Utility-first CSS framework for rapid development
- **No JavaScript**: Pure HTML/CSS implementation for better performance
- **Multiple Pages**: Complete website with 5+ pages
- **Search Functionality**: Integrated search bar in hero section
- **Contact Form**: Beautiful booking form with modern styling

## 📄 Pages

1. **Home (index.html)** - Main landing page with hero section, features, and popular dishes
2. **Menu (menu.html)** - Full menu showcase with categories and pricing
3. **About (pages.html)** - Restaurant story and chef profiles  
4. **Blog (blog.html)** - Latest news and culinary stories
5. **Shop (shop.html)** - Merchandise and specialty products

## 🎨 Design Elements

- **Header**: Sticky navigation with contact info and social links
- **Hero Section**: Full-screen hero with search bar and call-to-action buttons
- **Features**: Three-column layout highlighting quality, delivery, and recipes
- **Popular Dishes**: Grid showcase of signature menu items
- **Contact Section**: Split layout with contact info and booking form
- **Footer**: Complete footer with links and social media

## 🖼️ Images Folder

All images are organized in the `/images` directory including:
- Logo and branding assets
- Food photography
- Chef photos
- Blog images
- Product images

## 🚀 Getting Started

1. Open `index.html` in your web browser
2. No build process or dependencies required
3. All styling is handled via Tailwind CDN

## 🎨 Color Scheme

- **Primary**: Gold (#d4af37)

- **Text**: White/Gray tones

## 📱 Responsive Breakpoints

- Mobile: < 768px
- Tablet: 768px - 1024px
- Desktop: > 1024px

## 🔧 Technologies Used

## 📝 Note
